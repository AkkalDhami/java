# Day-03

## Variable

Container that store the data value.

## Datatype

### 1. Primitive

- **Integer**

  - `long`
    - width: 64 bits
    - range: -9,223,372,036,854,775,808 to 9,223,372,036,854,775,807

  - `int`
     - width: 32 bits
     - range: -2,147,483,648 to 2,147,483,647

  - `short`
     - width: 16 bits
     - range: -32,768 to 32,767

  - `byte`
     - width: 7 bits
     - range: -128 to 127

- **Real Number**

    - `double`
        - double precision
        - width: 64 bits
        - sin(), cos()

    - `float`
        - single precision
        - width: 32 bits
        - rarely used

- **Character**
    - `char`
         - 16 bit
         - store a single character
         - use single quotes
         - ex: 'a', 'Z'
         - char grade = 'B';

- **Boolean**
    - `boolean`
        - store _true_ or _false_
        - boolean isAdmin = false;
